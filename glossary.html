Glossary of Terms

A Computer Virus is a sub-type of Malware that infects Programs and Files present on the device and deploys some type of malicious Payload.

Types of Virus include:

  • Trojans - Often disguised as another program to trick users into downloading and opening it.
  • Worms - Does not infect system files, instead it self-replicates. Does not require user interaction to execute.
  • Browser Hijackers - Specifically infects Browsers, redirects affected Browsers to malicious websites.

Another type of Malware that spies on the infected device and logs all activity for transmission to an attacker. Often packaged with a Keylogger to record Keystrokes for Password harvesting purposes.

Also a sub-type of Malware. Instead of infecting the device and inflicting damage of some type it instead Encrypts the storage of the device. It would then demand payment in order to decrypt the device to restore it to working order.

A Cyber-Attack is any sort of malicious action carried out against digital devices. They can be performed over the Internet but also through physical devices such as USB drives.

A Cyber-Attacker can be an individual, group of individuals or even a Software Agent who carry out Cyber-Attacks.

A Network describes two or more devices that are linked together with some sort of data sharing infrastructure. Devices can be connected via Network on a Local basis, and a Network can be connected to the Internet. Which is itself a very large Network of Devices.

An action carried out with the intent to cause harm.

Stands for Standard Query Language. Its primary function is to access and manipulate Databases.

If an Information System is Vulnerable it means that it has some sort of weakness in its source code. If this vulnerability is exposed it can be used as an attack vector.
