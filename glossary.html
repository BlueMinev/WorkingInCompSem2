Glossary of Terms

A Computer Virus is a sub-type of Malware that infects Programs and Files present on the device and deploys some type of malicious Payload.

Types of Virus include:

  • Trojans - Often disguised as another program to trick users into downloading and opening it.
  • Worms - Does not infect system files, instead it self-replicates. Does not require user interaction to execute.
  • Browser Hijackers - Specifically infects Browsers, redirects affected Browsers to malicious websites.

Another type of Malware that spies on the infected device and logs all activity for transmission to an attacker. Often packaged with a Keylogger to record Keystrokes for Password harvesting purposes.

Also a sub-type of Malware. Instead of infecting the device and inflicting damage of some type it instead Encrypts the storage of the device. It would then demand payment in order to decrypt the device to restore it to working order.

A Cyber-Attack is any sort of malicious action carried out against digital devices. They can be performed over the Internet but also through physical devices such as USB drives.

A Cyber-Attacker can be an individual, group of individuals or even a Software Agent who carry out Cyber-Attacks.

A Network describes two or more devices that are linked together with some sort of data sharing infrastructure. Devices can be connected via Network on a Local basis, and a Network can be connected to the Internet. Which is itself a very large Network of Devices.

An action carried out with the intent to cause harm.

Stands for Standard Query Language. Its primary function is to access and manipulate Databases.

If an Information System is Vulnerable it means that it has some sort of weakness in its source code. If this vulnerability is exposed it can be used as an attack vector.

Internet of Things, abbreviated to IoT. This term describes any type of electronic device with integrated software that is capable of an Internet connection.

A Zombie in terms of Cybersecurity is used to describe a device infected with malicious code that is atleast partially under the control of an external agent. Usually used to perform actions such as DDoS Attacks.

When somebody is coerced it means that they have been compelled to act in an involuntary manner. Cyber-Attackers will often use threats or false propositions on their targets in order to coerce them into giving up sensitive information.

A payload in terms of Cybersecurity describes some sort of malicious code that is piggybacking on Malware. This can take many forms.

The code can be destructive, meaning it will cause harm to the device itself. It can be disruptive, meaning it will disrupt the operations of the device. It can also be passive, meaning it will sit in the background harvesting data from observed actions carried out on the device.

An Information system is an amalgamation of Hardware devices, Software Applications and Telecommunication Networks. Typically used to collect, store and analyse Data.

The most basic definition of Software is a set of instructions programmed to be readable and executable by a Computer. Modern Software applications will also processes user input, connect to various Hardware devices and communicate over available Networks.

Antivirus is a catch-all term used to describe Software speciallised in the detection of and protection from Malware. Modern Antivirus Software will usually also include other protection elements such as Firewalls and Intrusion detection and protection systems.

The term Device in terms of IT is used to describe a physical machine such as a Phone, Tablet, Computer, Integrated Circuit or Micro-controller.

Hardware loosely shares a definition with Device. However it is usually used to describe an individual component of a larger machine. For example, a Personal Computer contains Hardware components such as a Hard Disk Drive, Central Processing Unit and Random Access Memory modules.

Monitoring describes an action performed by a Software agent. Monitoring Software can be used to oversee and record actions taken on a device. Monitoring Software can be installed on a device as a protection method and it can also be used Maliciously if covertly installed by some Third-Party.

Traffic describes the flow of data across a Network. Traffic can be sent and received.

A Port describes a virtual point from where data is sent and received. Ports are numbered and each designation will typically be assigned to a certain Process or Service.

Logs are the product of a Monitoring application. They contain data about actions performed on a specific device. This includes: Date and Time, type of action taken and on what level the action was performed.

HTTP stands for Hyper-Text Transfer Protocol. It is the protocol used in the transfer of data over the Internet. HTTPS is the 'secure' version of HTTP it is the same but all data transferred is Encrypted.

A computer hacker is someone who tries to break into computer systems, especially in order to get secret information.

Phishing is the practice of trying to trick people into giving secret financial information by sending emails that look as if they come from a bank. The details are then used to steal people's money, or to steal their identity in order to commit crimes.

General Data Protection Regulation: European Union legislation introduced in 2018 with the aim of giving individuals more control over personal information held by companies.

In the U.S., HIPAA is an act that protects people covered by health insurance and makes rules about storing personal medical data.

To mitigate something means to make it less unpleasant, serious, or painful.

If something is unauthorized, it has been produced or is happening without official permission.

A sequence of characters used to authenticate a user during the login process.

An incident that results in unauthorised access to computer data.

A data set that provides information about other data but not the content, for example where and when a photo was taken.
